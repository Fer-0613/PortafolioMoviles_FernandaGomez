## 🗂️ Lista de Proyectos Incluidos

### 📊 [Proyecto U2: Análisis de Datos con Dart](./ProyectoU2_Analisis_Datos_Dart/)
* **Descripción:** Aplicación interactiva de consola capaz de decodificar datos complejos en formato JSON, manipular colecciones dinámicas y controlar valores nulos (Null Safety) para generar reportes estadísticos exportables.

### 🚀 [Proyecto U3: Tu Primera Aplicación de Flutter (my_awsome_namer)](./ProyectoU3_Primera_Aplicacion_Flutter/)
* **Descripción:** Introducción al desarrollo declarativo móvil. Una aplicación responsiva que utiliza un riel de navegación adaptativo (`LayoutBuilder`) para generar combinaciones aleatorias de palabras y administrar el estado global de favoritos mediante el patrón Provider.

### 🎵 [Proyecto U4: Mini Reproductor de Música (InLove)](./ProyectoU4_Mini_Reproductor_Musica/)
* **Descripción:** Aplicación multimedia avanzada con temática romántica neón. Implementa reproducción asíncrona de audio local mediante hilos (`Streams`), control estricto de memoria del hardware y transiciones visuales animadas de rotación de disco en tiempo real.

---

## 🧠 Competencias Desarrolladas

1. **Arquitectura Móvil:** Dominio del árbol de widgets estáticos y dinámicos (`Stateless` y `Stateful`).
2. **Gestión de Estados:** Sincronización eficiente de datos del sistema con la interfaz de usuario de forma reactiva (`Provider` y `Streams`).
3. **Diseño Responsivo:** Creación de maquetados flexibles y adaptables a múltiples resoluciones y orientaciones de pantalla.
4. **Calidad de Código:** Implementación de buenas prácticas de desarrollo bajo los estándares oficiales de Flutter Linting.
